ZAFIR YT WATCH LATER ORGANIZER

Personal YouTube Watch Later organizer with offline export, semantic clustering, and playlist repopulation.

Why this exists

YouTube’s Watch Later is a flat list without real categories. This project automates the boring part: export the full list, group videos by topic, then recreate organized private playlists in your own account.

Quick start

  1. Install dependencies: uv pip install google-api-python-client google-auth-oauthlib google-auth-httplib2 sentence-transformers scikit-learn
  2. Authorize YouTube account(s): python setup_auth.py --account 1 and python setup_auth.py --account 2
  3. Export Watch Later: python fetch_watch_later.py
  4. Build categories: python build_category_map.py
  5. Stage classification: python apply_categories.py
  6. Create YouTube playlists: python create_playlists.py

Tech stack

  • Python
  • Google OAuth 2.0
  • YouTube Data API v3
  • sentence-transformers
  • scikit-learn
  • yt-dlp

Privacy

This is a personal/local tool for one or two authenticated users. It does not collect telemetry.
