Privacy Policy

Effective date: 2026-06-20

Project: ZAFIR YT WATCH LATER ORGANIZER

Overview

This tool is a personal desktop application. It does not collect, transmit, or share personal data with any third party. All account data stays on the user’s own machine.

Data stored locally

  • OAuth credentials/tokens for the owner’s own YouTube account(s)
  • Exported Watch Later video lists
  • Generated classification and playlist mapping files

External services

  • Google OAuth 2.0 for YouTube account access
  • YouTube Data API v3 for playlist operations
  • Sentence-transformers model weights from Hugging Face for local embedding

No telemetry

The application does not phone home, use analytics, or transmit viewing data.
